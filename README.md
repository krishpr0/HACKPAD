- So What's this,

This is a Large macropad, I wanted to develop this for the unusal uses such as while gaming, editing, and production of the 3d models shorcuts. 

This operates on the Seeed Studio Xiao RP2040 DIP module with the QMK firmware.



## Hardware:- 

- MCU: Seeed XIAO RP2040 DIP
- Switches: 16x Cherry MX compatible Switches
- LED: 16x SK6812 RBG LED

# Programming,
- Its running the default firmware (QMK firmware) but you can update it to the system you want.

# BOM,

- 16x Cherry MX Switches, 16x SK6812 MINI LEDs, 1x XIAO RP2040, 16x Blank DSA Keycaps,  4x M3x16 Bolts,  4x M3 Heatset Inserts ~100g PLA
